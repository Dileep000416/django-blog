# Django Blog Application 🚀

A full-stack blog web application built using Django. This project allows users to register, log in, create posts, and manage content with authentication and authorization.

---

## 🔥 Features

* User Registration & Login System
* Authentication & Authorization
* Create, Read, and View Blog Posts
* Image Upload Support
* Slug-based URLs for SEO-friendly pages
* Admin Panel for managing posts
* Responsive UI using HTML, CSS, JS

---

## 🛠️ Tech Stack

* **Backend:** Python, Django
* **Frontend:** HTML, CSS, JavaScript
* **Database:** SQLite
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/django-blog.git
cd django-blog
```

2. Create virtual environment:

```bash
python -m venv .venv
```

3. Activate environment:

```bash
.venv\Scripts\activate
```

4. Install dependencies:

```bash
pip install -r requirements.txt
```

5. Apply migrations:

```bash
python manage.py migrate
```

6. Run the server:

```bash
python manage.py runserver
```

---

## 🔐 Admin Access

Create a superuser to access Django admin panel:

```bash
python manage.py createsuperuser
```

Then visit:

```
http://127.0.0.1:8000/admin/
```
